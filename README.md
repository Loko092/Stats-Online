# Stats-Online

Данное мини-приложение создана,чтобы узнавать и изучать свои статистики в разных играх(поддерживаются как мобильные игры как и ПК)



В нашем мини-приложение Вы можете выбрать свою игру, вписать свои: киллы,смерти,AVG(средние количество киллов за раунд)



Меню очень простое,Вам не придется изучать интерфейс этого приложения,все стоит на своих местах
