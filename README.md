простое и удобное мини-приложение для геймеров, которое помогает отслеживать количество смертей (киллов) в игровых сессиях и автоматически рассчитывать среднюю статистику. Идеально подходит для стримеров, киберспортсменов и всех, кто хочет анализировать свою эффективность в играх.

 😎Основные возможности
Подсчет смертей/киллов — быстрое добавление событий одним нажатием

Автоматический расчет AVG — среднее количество смертей за сессию/раунд

История сессий — сохранение данных 

Визуализация статистики — простые графики 

Экспорт данных — выгрузка в CSV/Excel для глубокого анализа
